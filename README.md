# Tourism Data Analysis Dashboard (Australia–Indonesia)

## Deskripsi Proyek
Proyek ini merupakan bagian dari kegiatan **Studi Independen Data Science** di **PT Vinix Seven Aurum**.  
Tujuan proyek adalah menyajikan analisis tren dan pola pariwisata Australia–Indonesia melalui dashboard visualisasi data interaktif berbasis **Tableau**.

Dashboard dirancang untuk membantu pemahaman data secara visual, mendukung analisis tren jangka panjang, pola musiman, serta perubahan mobilitas pariwisata berdasarkan data historis.

## Tujuan Proyek
- Menyajikan analisis tren pariwisata Australia–Indonesia berbasis data
- Mengidentifikasi pola musiman kunjungan wisatawan
- Menyajikan insight secara visual dan mudah dipahami
- Mendukung proses pembelajaran dan analisis data berbasis visualisasi

## Metodologi
Proyek ini mengadopsi pendekatan **CRISP-DM**, yang meliputi:
1. Business Understanding  
2. Data Understanding  
3. Data Preparation  
4. Data Analysis & Modeling  
5. Evaluation  
6. Deployment (Dashboard Visualization)

## Tools & Teknologi
- Python  
- Google Colab  
- Tableau  
- Microsoft Excel / CSV Dataset  

## Luaran Proyek
- Dashboard visualisasi data interaktif pariwisata Australia–Indonesia
- Insight tren jangka panjang dan pola musiman
- Dokumentasi analisis data sebagai bagian dari laporan akhir Studi Independen

## Hasil Singkat
Hasil analisis menunjukkan adanya variasi tren kunjungan wisatawan antarperiode serta pola musiman yang konsisten pada waktu tertentu. Dashboard memungkinkan pengguna mengeksplorasi data secara dinamis untuk memperoleh insight yang relevan.
